Games 4 Doge

A Contest to Promote the use of Dogecoin in Games!

Could Dogecin Become a Universal Game Currency?

If we help make it happen, I believe the answer is yes! (see latest updates here.)

The Game Contest

In the past, the Dogecoin community had a competition to build an Electrum-based wallet here, and we had some great results. Let's do it again, but this time around, build an awesome game! The only rule is it must incorporate Dogecoin in some way. The community will vote on the winners of each category. The ultimate goal of this competition is to explore innovative uses for the Dogecoin currency.

The Rules!

You must start a new project or integrate Dogecoin into an existing game. Existing games may win any category except the "most popular/viral" category which will go to a new game built in Oct 2014. The game may be a web app, or native platform app (iOS, Windows, Android..). All apps must be "secure" against theft of private data or dogecoins.

Example

See this Dogecoin space game faucet http://dt.htdt.ru/[2] , it's even open source! https://github.com/htdt/digital_trip[3]

Prize Categories!

The categories are especially designed to encourage games to be made. The following percentages of the pot will be assigned for each category in order of placement. (ie, first place in "most original" will get 10% of the pot)

  • Most popular/ viral. Must be a new game started during this contest timeframe, not a sequel. - 30% to winner, 10% to runner up.

  • Best game that incorporates Dogecoin as an in-game currency with in-game purchases. - 20% to winner.

  • Best game the integrates DogeParty somehow - 20% to winner

  • Most App Store downloads (web based games not eligible for this) - 10% awarded one month after contest completion.

  • Silliest / most ridiculous game - 10% to winner

The Due Date!

End of the day (PST) on October 31, 2014.

Guide for Developers

I suggest using block.io in order to integrate Dogecoin easily without running your own copy of dogecoind. If you're awarding Dogecoin to users for any type of in-game action, you'll need a way to verify they're not cheating. One straightforward way is with a server component such as socket.io in the example above. If anyone has more examples or resources for developers, please post them for everyone to benefit from! Here's some example code which deposit Dogecoin and spends it in-game. For an even simpler way to integrate Dogecoin, make your game a "faucet" that simply rewards your users when they get a high score by sending Dogecoin to their address.
