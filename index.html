A Dogecoin Game Contest!

TLDR: Build a game that uses Dogecoin in the next month, get 1 million Dogecoins!

I believe one great potentials, and "killer apps" if you will, for Dogecoin is to become the universal in-game currency. Whether or not, it's certainly true that one of the keys for us going to the moon is software infrastructure. If we build lots of popular apps that use dogecoin, the moon will follow in due course. For these reasons, ongoing discussion of an App contest has circulated for some time. For this contest, let's focus specifically on games. If it's a success (or not), a general app contest can always follow.

Dogecoin Game Contest

Once, we had a competition to build an Electrum-based wallet (http://www.reddit.com/r/dogecoin/comments/1upa5y/bounty_15_million_doge_for_the_first_person_to/[1] ), and we had some great results. Let's do it again, but this time around, build an awesome game! The only rule is it must incorporate Dogecoin in some way. The community will vote on the winners of each category. The ultimate goal of this competition is to explore innovative uses for the Dogecoin currency.

Rules

You must start a new project. You cannot submit something that was finished prior to the contest. However, you may submit a "sequel" to something pre-existing to any category except the "most popular/viral" category. The game may be a web app, or native platform app (iOS, Windows, Android..). All apps must be "secure" against theft of private data or dogecoins.

See this Dogecoin space game faucet http://dt.htdt.ru/[2] , it's even open source! https://github.com/htdt/digital_trip[3]

Categories

The categories are especially designed to encourage games to be made. The following percentages of the pot will be assigned for each category in order of placement. (ie, first place in "most original" will get 10% of the pot)

  • Most popular/ viral. Must be a new game started during this contest timeframe, not a sequel. - 30% to winner, 10% to runner up.

  • Best game that incorporates Dogecoin as an in-game currency with in-game purchases. - 20% to winner.

  • Best game the integrates DogeParty somehow - 20% to winner

  • Most App Store downloads (web based games not eligible for this) - 10% awarded one month after contest completion.

  • Silliest / most ridiculous game - 10% to winner

Due Date

End of the day (PST) on October 31, 2014.

Guide for Developers

I suggest using block.io in order to integrate Dogecoin easily without running your own copy of dogecoind. If you're awarding Dogecoin to users for any type of in-game action, you'll need a way to verify they're not cheating. One straightforward way is with a server component such as socket.io in the example above. If anyone has more examples or resources for developers, please post them for everyone to benefit from!
